## Full Description

MagicCarpet is a Paper plugin that adds a temporary flying carpet for players without permanently modifying the world.

The carpet follows the player in real time, supports smooth vertical movement, and is built to be safe for survival and public servers. Carpet blocks are temporary, protected, and restored immediately as the player moves.

### Features

- Toggleable magic carpet with `/mc`
- Default `5x5` platform
- Custom edge and center materials
- Optional special center block
- Smooth sneak descent
- Jump-based ascent
- Configurable movement values
- Protected temporary blocks
- No drops from carpet blocks
- Removes carpet instantly on logout
- Permission-based access

### Configuration

Server owners can customize:

- carpet size
- edge material
- center material
- center block usage
- ascent strength
- descent speed
- jump ascent duration
- plugin messages

### Permission

- `magiccarpet.use`

Default access is disabled, making the plugin suitable for VIP systems and premium ranks.
